
### Example Run (Basic Crawl)

```bash
python scraper.py --url "https://example.com"
```

### With Depth Control (crawl deeper)

```bash
python scraper.py --url "https://example.com" --depth 2
```

### With More Threads (faster parallel scraping)

```bash
python scraper.py --url "https://example.com" --threads 10
```

### With Embeddings (requires `sentence-transformers`)

```bash
python scraper.py --url "https://example.com" --embeddings
```

### Custom Output Filenames

```bash
python scraper.py --url "https://example.com" --data-file "output.json" --log-file "output_log.txt"
```

---

Eg:

>python app1.py --url "https://www.geeksforgeeks.org/machine-learning/bagging-vs-boosting-in-machine-learning/" --depth 1 --embeddings --data-file my_data.json --embeddings-file my_embeddings.json
